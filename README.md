# milistenna

Misskeyのリストとアンテナを管理するためのWebアプリケーションです。

## 概要

milistennaは、Misskeyインスタンスにおけるリストとアンテナの管理を簡単にするためのクライアントアプリケーションです。直感的なUIで、リストやアンテナの作成、編集、削除が可能です。

## 主な機能

### リスト管理
- リストの一覧表示
- モーダルによるリストの新規作成
- リストの編集（名前の変更、公開設定、メンバーの追加・削除）
- リストの削除
- リストメンバーの管理（ユーザー検索・追加・削除）
- 上限数の管理（インスタンスのポリシーに基づく）

### アンテナ管理
- アンテナの一覧表示
- 専用フォームによるアンテナの新規作成
- アンテナの編集
- アンテナの削除
- アンテナのコピー（既存アンテナの設定を複製して新規作成）
- ソース設定（すべての投稿、指定ユーザー、指定ユーザーを除外、リスト）
- キーワード設定（AND/OR条件）
- 詳細オプション（ローカルのみ、大文字小文字区別、Bot除外、リプライ含む、ファイル付きのみ等）
- 上限数の管理（インスタンスのポリシーに基づく）

### 認証
- MiAuth認証による安全なログイン
- 任意のMisskeyインスタンスに対応
- ローカルストレージによるセッション永続化
